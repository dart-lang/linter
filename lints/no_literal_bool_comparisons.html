Using the Linter

From Effective Dart:

DON'T use true or false in equality operations.

This lint applies only if the expression is of a non-nullable bool type.

BAD:

if (someBool == true) {
}
while (someBool == false) {
}

GOOD:

if (someBool) {
}
while (!someBool) {
}
