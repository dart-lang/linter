Using the Linter

PREFER declaring const constructors on @immutable classes.

If a class is immutable, it is usually a good idea to make its constructor a const constructor.

BAD:

@immutable
class A {
  final a;
  A(this.a);
}

GOOD:

@immutable
class A {
  final a;
  const A(this.a);
}

Usage

To enable the prefer_const_constructors_in_immutables lint, add prefer_const_constructors_in_immutables under linter > rules in your analysis_options.yaml file:

linter:
  rules:
    - prefer_const_constructors_in_immutables
