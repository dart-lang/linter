Using the Linter

DO put asserts in initializer lists.

BAD:

class A {
  A(int a) {
    assert(a != 0);
  }
}

GOOD:

class A {
  A(int a) : assert(a != 0);
}

Usage

To enable the prefer_asserts_in_initializer_lists lint, add prefer_asserts_in_initializer_lists under linter > rules in your analysis_options.yaml file:

linter:
  rules:
    - prefer_asserts_in_initializer_lists
