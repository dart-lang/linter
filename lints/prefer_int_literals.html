Using the Linter

DO use int literals rather than the corresponding double literal.

BAD:

const double myDouble = 8.0;
final anotherDouble = myDouble + 7.0e2;
main() {
  someMethod(6.0);
}

GOOD:

const double myDouble = 8;
final anotherDouble = myDouble + 700;
main() {
  someMethod(6);
}
