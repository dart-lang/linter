Using the Linter

DON'T create recursive getters.

Recursive getters are getters which return themselves as a value. This is usually a typo.

BAD:

int get field => field; // LINT

BAD:

int get otherField {
  return otherField; // LINT
}

GOOD:

int get field => _field;

Usage

To enable the recursive_getters lint, add recursive_getters under linter > rules in your analysis_options.yaml file:

linter:
  rules:
    - recursive_getters
