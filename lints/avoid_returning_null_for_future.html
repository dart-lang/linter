Using the Linter

AVOID returning null for Future.

It is almost always wrong to return null for a Future. Most of the time the developer simply forgot to put an async keyword on the function.

Usage

To enable the avoid_returning_null_for_future lint, add avoid_returning_null_for_future under linter > rules in your analysis_options.yaml file:

linter:
  rules:
    - avoid_returning_null_for_future
