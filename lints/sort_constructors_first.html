Using the Linter

DO sort constructor declarations before other members.

GOOD:

abstract class Animation<T> {
  const Animation(this.value);
  double value;
  void addListener(VoidCallback listener);
}

BAD:

abstract class Visitor {
  double value;
  visitSomething(Something s);
  Visitor();
}
