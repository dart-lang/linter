Using the Linter

DO put a single newline at the end of non-empty files.

BAD:

a {
}

GOOD:

b {
}
    <-- newline

Usage

To enable the eol_at_end_of_file lint, add eol_at_end_of_file under linter > rules in your analysis_options.yaml file:

linter:
  rules:
    - eol_at_end_of_file
