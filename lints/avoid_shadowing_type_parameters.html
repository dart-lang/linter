Using the Linter

AVOID shadowing type parameters.

BAD:

class A<T> {
  void fn<T>() {}
}

GOOD:

class A<T> {
  void fn<U>() {}
}

Usage

To enable the avoid_shadowing_type_parameters lint, add avoid_shadowing_type_parameters under linter > rules in your analysis_options.yaml file:

linter:
  rules:
    - avoid_shadowing_type_parameters
