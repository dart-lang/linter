prefer_bool_in_asserts

Group: style

Maturity: removed

Using the Linter

Unsupported since Dart language version 3.0.0.

DO use a boolean for assert conditions.

Not using booleans in assert conditions can lead to code where it isn't clear what the intention of the assert statement is.

BAD:

assert(() {
  f();
  return true;
});

GOOD:

assert(() {
  f();
  return true;
}());

Usage

To enable the prefer_bool_in_asserts lint, add prefer_bool_in_asserts under linter > rules in your analysis_options.yaml file:

linter:
  rules:
    - prefer_bool_in_asserts
