Using the Linter

DO join return statement with assignment when possible.

BAD:

class A {
  B _lazyInstance;
  static B get instance {
    _lazyInstance ??= B(); // LINT
    return _lazyInstance;
  }
}

GOOD:

class A {
  B _lazyInstance;
  static B get instance => _lazyInstance ??= B();
}

Usage

To enable the join_return_with_assignment lint, add join_return_with_assignment under linter > rules in your analysis_options.yaml file:

linter:
  rules:
    - join_return_with_assignment
